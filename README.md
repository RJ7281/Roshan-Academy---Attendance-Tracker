📑 Roshan Academy - Attendance Tracker
A modern, cloud-based attendance management system built for Roshan Academy. This application allows teachers to securely log in, select specific standards (classes), and sync student attendance records in real-time using Firebase.

✨ Features
🔒 Secure Access: Protected by a teacher-specific master password with a "show/hide" toggle for better UX.

☁️ Real-time Sync: Powered by Firebase Realtime Database to ensure data is saved and retrieved instantly across devices.

🎨 Modern UI/UX: A clean "Glassmorphism" interface built with Tailwind CSS, featuring:

Responsive design for mobile and desktop.

Live row highlighting (red for absent, green for present).

Micro-animations for button clicks and screen transitions.

📊 Multi-Standard Support: Separate student databases for Standards 1 through 10.

📅 Date-Specific Tracking: Easily log or view attendance for any specific calendar date.

🛠️ Tech Stack
Frontend: HTML5, JavaScript (ES6 Modules)

Styling: Tailwind CSS (via CDN)

Icons: FontAwesome 6

Backend/Database: Firebase Realtime Database
